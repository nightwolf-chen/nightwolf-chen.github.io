是时候回到GitHub的怀抱了

又到了折腾的季节，jekyll搞得我蛋疼，Farbox已经让我蛋碎了。 Farbox的垃圾同步机制，没有版本管理, 把我的文章都搞没了，诶。 最好还是自己手握所有文章并且用版本管理才是王道啊。本来想回归Jekyll的， 我只想写写文章，不想重新学一门语言！Hugo看起来方便一些，先用用看吧。

回到GitHub的感觉就跟回家一样！
